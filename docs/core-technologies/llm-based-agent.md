# LLM 기반 에이전트

LLM 기반 에이전트는 대규모 언어 모델을 두뇌로 활용하여 복잡한 작업을 수행하는 시스템입니다.

## 아키텍처 구성

### 핵심 LLM

에이전트의 중앙 처리 장치 역할을 수행합니다:

- 자연어 이해 및 생성
- 추론과 계획 수립
- 도구 선택 및 호출 결정

### 프롬프트 엔지니어링

에이전트의 행동을 정의하는 핵심 요소입니다:

- **시스템 프롬프트**: 에이전트의 역할과 제약 조건 정의
- **Few-shot 예제**: 원하는 행동 패턴 시연
- **Chain-of-Thought**: 단계별 추론 유도

### 도구 통합

외부 기능을 활용하여 LLM의 한계를 극복합니다:

- 웹 검색 및 브라우징
- 코드 실행 환경
- 데이터베이스 접근
- API 호출

## 주요 프레임워크

### LangChain

- 다양한 LLM과 도구를 연결하는 프레임워크
- 체인 기반의 워크플로우 구성

### AutoGPT / AgentGPT

- 자율적 목표 달성을 위한 에이전트 시스템
- 반복적 계획-실행-평가 사이클

## ReAct 패턴

Reasoning과 Acting을 결합한 접근 방식:

1. **Thought**: 현재 상황 분석
2. **Action**: 적절한 도구 선택 및 실행
3. **Observation**: 결과 확인
4. **반복**: 목표 달성까지 순환
