# 무기 인간(하이브리드)의 생태 및 특성

심장을 악마와 융합하여 불사의 신체와 무기 변신 능력을 얻은 '무기 인간'의 생물학적 메커니즘을 설명합니다.

## 1. 개요

**무기 인간(Hybrid)**이란 인간의 심장을 악마의 심장으로 대체하거나, 특수한 계약을 통해 악마와 일체화된 존재를 의미합니다. '마인(Fiend)'과 달리 인간의 자아를 완벽하게 유지하며, 평상시에는 인간과 구별할 수 없는 외형을 가집니다. 이들은 특정 조건을 만족하면 신체 일부 혹은 전신을 무기 형태로 변형하여 강력한 파괴력을 행사할 수 있습니다.

## 2. 핵심 메커니즘: 트리거(Trigger)

모든 무기 인간은 변신을 시작하기 위한 고유한 '트리거'를 가지고 있습니다. 이 트리거는 신체 외부에 노출된 특정 부위를 조작함으로써 활성화됩니다. 트리거가 당겨지면 내장된 악마의 힘이 분출되며 외형이 변화합니다.

### 주요 트리거 예시
- **시동 로프**: 가슴 부위에 돌출된 엔진 줄을 당김 (예: 전기톱)
- **안전핀**: 목 부위의 핀을 뽑음 (예: 폭탄)
- **신체 부착물**: 손목의 특정 부위를 당기거나 머리의 부속물을 조작함

```csharp
// 무기 인간 변신 로직의 개념적 코드
public class HybridHuman : Entity
{
    public bool IsTransformed = false;
    public float BloodLevel = 100.0f;

    public void ActivateTrigger()
    {
        if (BloodLevel > 10.0f)
        {
            this.IsTransformed = true;
            this.ApplyTransformationEffect();
            this.RegenerateHealth(100.0f);
        }
    }
}
```

## 3. 신체적 특성 및 재생 능력

무기 인간의 가장 큰 특징은 **반불사성**입니다. 이들은 일반적인 인간이나 악마와는 궤를 달리하는 생존력을 보유합니다.

1. **초재생 능력**: 팔다리가 잘리거나 머리가 파괴되어 시체와 다름없는 상태가 되어도, 트리거를 당기면 즉시 모든 신체가 재생됩니다.
2. **혈액 의존성**: 모든 재생과 변신에는 막대한 양의 혈액이 소모됩니다. 혈액이 부족하면 변신이 풀리거나 재생이 중단될 수 있습니다.
3. **심장의 영속성**: 악마와 융합된 '심장' 자체가 파괴되지 않는 한, 사실상 노화하지 않으며 영구적인 생명을 누립니다.

## 4. 전투 형태 및 효과

변신 시 사용자는 다음과 같은 전투적 이득을 얻습니다.

- **물리적 강화**: 근력, 내구도, 반응 속도가 인간의 범주를 초월하여 상급 악마와 대등하게 싸울 수 있습니다.
- **고유 무장**: 신체에서 전기톱, 폭탄, 검, 활 등 고유한 무기를 생성하여 공격합니다.
- **지속성**: 적을 타격하여 흘러나오는 혈액을 섭취함으로써 전투 중 실시간으로 상처를 치유하고 변신 시간을 연장할 수 있습니다.

> 💡 **시스템 노트**: 무기 인간은 존재 자체가 모순적이며, '체인소 맨'에 의해 그 이름이 지워졌음에도 불구하고 존재가 유지되는 유일한 예외적 부류입니다. 이들은 공안이나 민간 데빌 헌터 조직에서 전략 병기로 활용되기도 합니다.
