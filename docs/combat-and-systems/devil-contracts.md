# 악마의 계약 규칙

인간과 악마 사이의 위험한 권능 거래 프로세스와 그에 따른 대가 산정 방식, 그리고 위반 시의 금기 사항을 분석합니다.

## 1. 개요

**악마의 계약(Demon Pact)**은 필멸자가 초월적인 힘을 얻기 위해 자신의 영혼, 생명력, 혹은 감정을 담보로 고위 악마와 맺는 초자연적 결속입니다. 이 시스템은 '등가교환'의 법칙을 따르며, 계약자의 욕망이 클수록 지불해야 할 대가 또한 기하급수적으로 상승합니다.

## 2. 계약 체결 프로세스

계약은 통상적으로 다음의 3단계 공정을 거쳐 완성됩니다.

1.  **소환 및 매개체 설정**: 고유한 문양(Sigil)과 매개물(피, 유골 등)을 통해 악마의 의식을 현세로 초빙합니다.
2.  **교섭(Negotiation)**: 악마가 부여할 권능의 범위와 계약자가 지불할 대가를 확정합니다. 이때 계약의 세부 조항은 서술적 논리로 구성됩니다.
3.  **각인(Imprinting)**: 계약자의 신체 부위에 '낙인'이 새겨지며, 이는 악마의 마력이 흐르는 통로이자 구속의 상징이 됩니다.

## 3. 대가 지불 메커니즘

대가는 크게 '고정 비용'과 '운영 비용'으로 구분됩니다.

### 3.1 고정 비용 (Fixed Cost)
계약 체결 시 즉시 지불하는 가치입니다.
-   **신체 일부**: 시력, 청력, 혹은 팔다리의 기능 상실.
-   **기억 및 감정**: 가장 소중한 기억이나 '사랑', '공포'와 같은 특정 감정의 영구적 결여.

### 3.2 운영 비용 (Operating Cost)
권능을 활성화할 때마다 소모되는 자원입니다.

```json
{
  "resource_type": "Vitality",
  "consumption_rate": "0.5% per second",
  "critical_threshold": "Below 10% HP leads to Soul Possession"
}
```

## 4. 금기 사항 (Taboos) 및 페널티

계약 체결 후 다음 금기 사항을 어길 시, 계약은 즉시 파기되며 악마에게 영혼의 전권을 박탈당합니다.

-   **성역 침범**: 신성 성향의 마력이 깃든 장소(성당, 결계 등)에 동의 없이 진입하는 행위.
-   **이중 계약**: 동일한 영혼을 담보로 다른 고위 존재와 추가 계약을 맺는 행위.
-   **계약 주체 살해 시도**: 계약을 맺은 악마를 물리적으로 제거하여 대가를 회피하려는 시도.

## 5. 실용적 예시: '심연의 눈' 계약

이 계약은 전장의 흐름을 읽는 초월적 통찰력을 제공합니다.

-   **제공 권능**: 모든 은신 상태 해제 및 적의 약점 자동 포착.
-   **지불 대가**: 계약자의 왼쪽 눈의 시력을 영구히 상실하며, 권능 사용 시마다 1분간 극심한 고통(Neural Shock) 유발.
-   **금기**: 권능을 사용하는 동안 절대 '거짓말'을 해서는 안 됨. 위반 시 즉시 계약자의 심장 정지.
