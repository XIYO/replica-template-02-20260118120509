# 공안 대마 특이과 (Public Safety Devil Hunters Special Division)

국가 내각관방 직속의 데빌 헌터 조직인 공안 대마 특이과의 운영 체계와 주요 인물 구성을 다룹니다.

## 개요

**공안 대마 특이과**는 국가 공공 안전을 위협하는 악마(Devil)를 퇴치하기 위해 설립된 내각관방 직속의 특수 기관입니다. 민간 데빌 헌터가 해결하기 어려운 고위험군 악마를 추적하고 격리하는 것을 주 목적으로 하며, 일반적인 경찰이나 군대보다 강력한 법적 권한과 특수 장비를 보유하고 있습니다.

## 운영 체계 및 특징

공안 대마 특이과는 일반적인 수사 기관과 차별화된 다음과 같은 운영 특징을 가집니다.

### 1. 악마와의 계약 시스템
대마 특이과 대원들은 악마를 효율적으로 상대하기 위해 스스로 악마와 계약을 맺습니다. 계약의 대가는 개인마다 다르며, 다음과 같은 자원을 소모합니다.
- **수명**: 특정 기술 사용 시 잔여 수명을 공여함
- **신체 부위**: 감각(미각, 시각 등)이나 장기의 일부를 제물로 바침
- **미래**: 자신의 미래에 대한 권리를 양도함

### 2. 마인(Fiend) 및 악마의 전략적 배치
특이 4과와 같은 실험적 부대에서는 지능을 가진 마인이나 인간에게 협력적인 악마를 직접 대원으로 기용합니다. 이는 인명 피해를 줄이고 악마 특유의 비정상적인 능력을 역이용하기 위함입니다.

### 3. 법적 권한 및 예산
국가 예산으로 운영되며, 작전 중 발생하는 기물 파손이나 민간인 피해에 대해 폭넓은 면책 특권을 가집니다. 또한 일반 경찰이 접근할 수 없는 초자연적 사건 현장에 대한 우선권을 보유합니다.

## 주요 구성원 (제4과 중심)

| 이름 | 역할 | 계약 악마 / 정체 | 비고 |
| :--- | :--- | :--- | :--- |
| **마키마** | 총 책임자 | 지배의 악마 | 내각관방 장관 직속의 실권자 |
| **하야카와 아키** | 현장 리더 | 여우, 커스, 미래의 악마 | 공안 소속의 베테랑 헌터 |
| **덴지** | 전방 공격수 | 체인소의 악마 (포치타) | 악마의 심장을 가진 반인반마 |
| **파워** | 현장 요원 | 피의 마인 | 전직 피의 악마, 제멋대로인 성격 |
| **키시베** | 지휘관/스승 | 발톱, 칼날, 바늘의 악마 | '최강의 데빌 헌터'라 불리는 인물 |

## 작전 프로토콜 예시

공안 대원들이 현장에 투입되었을 때 따르는 기본적인 논리 구조는 다음과 같습니다.

```pseudocode
Function DevilElimination(target_devil):
    // 1. 위협 수준 평가
    threat_level = Analyze(target_devil)
    
    // 2. 적합한 계약 능력 선택
    If threat_level > HIGH:
        UseContractAbility("Future Devil", "Prediction")
    Else:
        DeployFiends(Denji, Power)

    // 3. 교전 및 포획/사살
    Combat(target_devil)
    
    // 4. 사후 처리
    If CivilianWitnesses > 0:
        ApplyMemoryManipulation() // 지배의 능력 또는 약물 사용
    ReportStatusToCentral()
End Function
```

## 결론

공안 대마 특이과는 악마라는 비상식적인 위협에 맞서기 위해 스스로 비상식적인 수단을 동원하는 조직입니다. 대원들은 높은 사망률과 가혹한 계약 조건 속에서도 국가의 안녕을 위해 암약하고 있습니다. 이들의 활동은 철저히 비밀에 부쳐지며, 실패의 대가는 곧 대원의 죽음이나 그 이상의 파멸로 직결됩니다.
