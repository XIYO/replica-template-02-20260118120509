# 지배의 악마와 묵시록의 4기사

세상의 근간을 뒤흔드는 네 마리의 강력한 악마인 묵시록의 4기사의 개념과 각 개체의 특성을 정의합니다.

## 개요

**묵시록의 4기사(Four Horsemen)**는 인류가 태초부터 품어온 가장 근원적인 공포인 지배, 전쟁, 기근, 죽음을 상징하는 네 마리의 강력한 악마를 일컫는 명칭입니다. 이들은 일반적인 악마들과 달리 인간에 가까운 형태를 유지하는 경우가 많으며, 세계의 질서를 재편할 수 있는 초월적인 권능을 보유하고 있습니다.

## 공통 특성

4기사는 다른 악마들과 구별되는 몇 가지 고유한 시스템적 특징을 공유합니다.

- **인식의 눈**: 동공에 여러 개의 동심원이 그려진 특유의 눈을 가지고 있습니다.
- **개념적 불사**: 이들이 상징하는 공포가 사라지지 않는 한, 지옥과 현세를 오가며 끊임없이 윤회합니다.
- **초월적 기억**: '체인소 맨'에게 먹혀 세상에서 사라진 존재들의 이름을 기억할 수 있는 소수의 존재들입니다.

## 4기사 상세 프로필

### 1. 지배의 악마 (Control Devil)
타인을 지배하고 복종시키는 힘을 가집니다. 자신의 능력을 과신하며, 하등하다고 판단되는 존재의 의지를 찬탈합니다.

- **핵심 능력**: 자신보다 하등하다고 판단되는 대상을 조종
- **계약 특징**: 다수의 하위 계약자를 통해 피해를 전가하거나 능력을 공유함

### 2. 전쟁의 악마 (War Devil)
자신이 소유권을 가졌다고 생각하는 대상을 강력한 무기로 변환합니다.

- **핵심 능력**: 대상을 무기화(Weaponization)
- **위력 결정 요소**: 대상을 무기로 만들 때 느끼는 '죄책감'의 크기에 비례하여 화력이 강화됨

### 3. 기근의 악마 (Famine Devil)
굶주림을 매개로 타자를 속박합니다. 감정적, 물리적 굶주림을 느끼는 존재를 자신의 '말'로 부립니다.

- **핵심 능력**: 굶주린 존재의 소환 및 통제
- **전략**: 대상의 욕구를 결핍시켜 강제적으로 지배 하에 둠

### 4. 죽음의 악마 (Death Devil)
4기사 중 가장 먼저 태어났으며, 생명체가 가진 가장 거대한 공포인 '죽음' 그 자체를 상징합니다.

- **특이사항**: 나머지 세 자매가 힘을 합쳐야 대적할 수 있을 정도의 압도적인 위상을 가짐

## 능력 발현 예시 (Pseudo-Code)

4기사의 권능이 시스템적으로 어떻게 작동하는지 설명하기 위한 예시 코드입니다.

```javascript
// 지배의 악마: 지배 로직 예시
function controlTarget(user, target) {
  if (user.perception.isInferior(target)) {
    target.status = "CONTROLLED";
    target.owner = user.id;
    return "지배에 성공했습니다.";
  }
  return "지배에 실패했습니다.";
}

// 전쟁의 악마: 무기화 로직 예시
function createWeapon(target, guiltLevel) {
  const baseDamage = target.value * 10;
  const finalDamage = baseDamage * (1 + guiltLevel);
  return new Weapon(target.name + " Sword", finalDamage);
}
```

## 결론 및 주의사항

묵시록의 4기사는 개별적으로도 재앙에 가까운 파괴력을 지니지만, 이들이 서로 협력하거나 대립할 때 발생하는 여파는 세계 멸망의 트리거가 될 수 있습니다. 특히 이들은 서로를 자매라 부르면서도 각자의 목적을 위해 서로를 이용하거나 배신하는 복잡한 관계망을 형성하고 있으므로, 이들의 행보를 관찰할 때는 단순한 선악의 구도를 벗어나 이해해야 합니다.
